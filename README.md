# Android Keyboard Factory

Android端末単体の操作だけで、自分専用のキーボードアプリ（IME）を作って、APKとしてDownloadsに書き出せるアプリ。PCもサーバーも使わず、端末上でキー配列を組んでそのままインストール可能なアプリを生成する。

## できること

- **キーボードプロジェクトの管理**: 名前を付けて複数のキーボードを作成・検索・削除できる（SQLite/Room管理）。
- **グリッドエディタ**: ページごとに縦横のキー数（最大32×32）を設定し、キーごとに役割（送信する文字/Enter/Delete/次ページ・前ページへの移動/何もしない）を割り当てられる。
- **キーの結合**: 隣接するキーを上下左右に結合して、Enterキーを大きくしたり階段状・L字型など自由な形のキーを作れる。
- **フォント**: 端末内の`.ttf`/`.otf`ファイルを取り込み、プロジェクト全体のデフォルト＋ページごとの上書きとして設定できる。エクスポート時にキーの文字を画像化するので、生成アプリにフォント丸ごとを含めずに済む。
- **APK書き出し**: 作成したキーボードを、実際にインストールできるAPKとしてDownloadsフォルダへ出力する。プロジェクトごとに固定のアプリ名・パッケージ名・署名鍵を保持するので、編集してから再書き出しすれば「アプリの更新」として上書きインストールできる。

## ディレクトリ構成

```
:app                  ファクトリー本体（プロジェクト管理・エディタUI・フォント取込・
                       ラスタライズ・APK生成パイプライン）
:keyboard-engine       :app と :keyboard-template の両方が使う共有部分
                       （キーボードのレイアウト定義・グリッド描画・結合ロジック）
:keyboard-template     生成されるキーボードアプリの素になる、最小限のIME本体
                       （:app のビルド時にコンパイルされ、:app のassetsに同梱される）

docs/<year>/<month>/   設計・実装の経緯や決定事項、トラブルシューティングの記録
tmp/                   ビルドログ等の作業用スクラッチ（gitignore対象、./clean_tmp.shで初期化）
build.sh               ビルド/インストール用スクリプト
```

開発・ビルド時の注意点や既知のハマりどころは[CLAUDE.md](CLAUDE.md)、設計の背景や経緯は`docs/`配下を参照。
